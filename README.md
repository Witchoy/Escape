# Escape

A first-person escape room game built in Unity 6. Currently a work in progress.

## Gameplay

You are locked in a room and must find items, solve a combination padlock, and escape. 

- Examine objects up close with the inspect system
- Pick up and manage items in your hotbar
- Use items on interactables (keys on doors, etc.)
- Crack the padlock combination to unlock the chest and escape

## Features

- First-person movement with walk, run, crouch, and jump
- Hotbar inventory with stackable items
- Item pickup, drop, and use system
- Inspect mode to examine items in detail
- Combination padlock with rotating number wheels
- Door and chest interaction
- Audio mixer with separate master, music, and SFX volume controls
- Animated scene transitions across three scenes (menu → game → end)
- Item highlight on hover

## Controls

| Action | Input |
|---|---|
| Move | WASD |
| Look | Mouse |
| Sprint | Shift |
| Crouch | C |
| Jump | Space |
| Grab / Interact | E |
| Use held item | Left click |
| Inspect | I |
| Drop | Q |
| Hotbar select | 1–4 |

All controls can be edited
